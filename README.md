# Songic

A Flutter music app made with Provider and BLoC pattern.

Recommended Flutter version: <b>1.7.8+hotfix.3</b>

## Current Features
[✓] Retrieve all songs

[✓] Pause

[✓] Play

[✓] Seek

[✓] Repeat

[✓] Shuffle

[✓] Now Playing

[✓] add to favorites

[✓] Search songs

[✓] Albums

[✓] Well-designed UI
