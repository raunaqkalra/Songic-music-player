# Songic

A Flutter music app made with Provider and BLoC pattern.


## Current Features
1. Retrieve all songs

2. Pause

3. Play

4. Seek

5. Repeat

6. Shuffle

7. Now Playing

8. add to favorites

9. Search songs

10. Albums

11. Well-designed UI
